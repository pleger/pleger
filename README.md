# Hi, I'm Paul Leger👋
<small>[https://pleger.cl](https://pleger.cl) </small>


**Associate Professor · Software Engineer Researcher · Lab Director**  
Catholic University of the North — Coquimbo, Chile 🇨🇱

I work at the intersection of **programming languages**, **software engineering**, and **agent-based modeling**. My research focuses on building better abstractions for developers — from aspect-oriented and context-oriented programming to back-in-time debuggers and ecosystem evolution studies.

I lead the [Pragmatics Lab](https://pragmaticslab.com), where we build tools and run studies that make software development more expressive, modular, and understandable.

---

## 🔬 Research Interests

- **Programming Languages** — modularity, abstractions, functional & context-oriented programming
- **Software Engineering** — developer tools, debugging, language ecosystems
- **Agent-Based Models** — simulating social behaviors on platforms like Twitter/X and e-commerce
